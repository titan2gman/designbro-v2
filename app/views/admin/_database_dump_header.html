1. 1. Download database dump via Download button

  2. 2. In terminal move to your rails project directory 

          cd your/rails_app_directory/

  3. 3. Shut down rails server if it is running

  4. 4. Remove and create local database:

    You should be in rails project directory 
          bundle exec rake db:drop db:create

  5. 5. Importing data from dumped database:

          gunzip <path_to_downloaded_file_name>/filename.gz | psql -U <local_database_user_name> -d designbro_development -f <path_to_downloaded_file_name>/filename

    Get local_database_user_name from .env file. See DB_USERNAME column

    For example, if database dump was downloaded into <rails_project>/tmp

    and you are in <rails_project> directory

    and downloaded file name is some_file_name.gz

    and DB_USERNAME is db_username

    you should do next:

          gunzip tmp/some_file_name.gz | psql -U db_username -d designbro_development -f tmp/some_file_name
